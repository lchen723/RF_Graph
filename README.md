This online supplementary material contains R scripts for simulation and data analysis in the manuscript "Predictor-Assisted Nonparametric Graphical Models with Multivariate Error-Prone Data".
1. RDA_SiM_v2.R: An R script for running GBM data analysis.
2. RF_Sim_compare_ME.R: An R script for running simulation studies and examining impacts of measurement error.
3. function_NP_Graph.R: An R script containing the function for the method in the manuscript.
